# Sources images (non servies)

Ce dossier contient des PNG sources qui ne sont PAS references par le site en
production (verifie le 18/07/2026 : aucun lien dans les .html, .css ou .js).
Ils sont conserves ici comme sources, hors du dossier servi img/, pour alleger
img/ sans rien supprimer.

Contenu : les images brutes generees (ChatGPT Image ...png) et deux anciens
originaux remplaces par des WebP servis (banner-hero.png remplace par
hero-*.webp ; ParrotTalk_logo_contact.png remplace par logo-official.webp).

Le seul PNG reellement servi (og-image.png, apercu social) reste dans img/.
Option de suivi possible : ajouter ce dossier a .vercelignore pour l'exclure
aussi du deploiement (non fait ici : aucune modif de config de deploiement).
